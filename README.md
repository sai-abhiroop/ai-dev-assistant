# 🚀 AI Developer Assistant

An AI-powered coding assistant with authentication, OTP verification, Google login, and code analysis.

## ✨ Features
- 🔐 Login / Signup (JWT + OTP)
- 🌐 Google Authentication
- 🤖 AI Code Explanation, Debugging, Optimization
- 🧠 Chat History
- 💻 Monaco Code Editor

## 🛠 Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- AI: Gemini API

## 🚀 Deployment
- Frontend: Vercel
- Backend: Render
